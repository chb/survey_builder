About Sections:
  • Sections are reusable collections of Questions and Answers
  • Once you create a Section, you can drag and drop into your Survey or another Section. This will add a branch to the dragged Section and its elements
  • If a Section is not branched to from your Survey, its contents will not appear in the Survey
About Questions:
  • Quesions allow you to present the user with question text followed by one or more Answers
  • Drag and drop a Question onto your Survey or a Section to add it
Question Types:
  • Simple - Present the user with a question that requires them to enter a single answer into a box
  • Multiple Choice - Present the user with a question that allows them to choose a single option from a list of Answers
  • All that Apply - Present the user with a question that allows them to choose zero or more options from a list of Answers
About Answers:
  • Answers allow you to present the user with an option to choose for the enclosing Question
  • Drag and drop an Answer onto an existing Question to add it
  • You can also quick add an Answer from an existing Question by hovering your mouse over the plus sign and selecting the Answer type you wish to add
Answer Types:
  • Free Text - Allow the user to input their answer into a box
  • Fixed Choice - Shows a set option the user can choose
About Logic Components:
  • Logic Components allow you to add increased control over the path a user takes through your survey
  • Drag and drop a Logic Component onto your Survey or a Section to add it
Logic Component Types:
  • Conditional Branch - Branch to a Section of Questions based off of an answer to a previous question
